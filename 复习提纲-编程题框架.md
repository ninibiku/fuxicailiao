## 一、Pandas 数据处理（第 1—8 题）——5 种高频模式

### 模式 ①：读取 → 清洗 → 分组聚合 → 保存

> 覆盖题号：1、2、4、6

**核心框架：**
```
读文件 → 类型转换/去重/过滤 → groupby 聚合 → 保存结果
```

| 步骤 | 必记函数 |
|------|----------|
| 读 CSV | `pd.read_csv('xxx.csv')` |
| 读 Excel | `pd.read_excel('xxx.xlsx')` |
| 日期类型转换 | `pd.to_datetime(df['日期'])` |
| 提取月份 | `df['日期'].dt.month` |
| 去重 | `df.drop_duplicates()` / `df.drop_duplicates(subset=['列'])` |
| 条件过滤 | `df[df['列'] != '值']` |
| 分组聚合 | `df.groupby('列')['数值列'].sum() / .mean() / .agg(['sum','mean'])` |
| 排序 | `df.sort_values('列', ascending=False)` |
| 保存 CSV | `df.to_csv('xxx.csv', index=False, encoding='utf-8-sig')` |

**极简框架（以题1为例）：**
```python
import pandas as pd
df = pd.read_excel('sales_data.xlsx')        # 读
df['日期'] = pd.to_datetime(df['日期'])       # 转换
df['月份'] = df['日期'].dt.month
result = df.groupby('月份')['销售额'].sum()    # 聚合
result.to_csv('monthly_sales_result.csv')     # 保存
```

### 模式 ②：时间序列处理

> 覆盖题号：2

**与普通分组的关键区别：用 `resample` 替代 `groupby`**

**核心框架：**
```python
df['日期'] = pd.to_datetime(df['日期'])
df.set_index('日期', inplace=True)                              # 设为时间索引
result = df['数值列'].resample('M').agg(['sum', 'mean'])        # 按月聚合
```

| 重采样频率 | 含义 |
|-----------|------|
| `'M'` | 按月 |
| `'W'` | 按周 |
| `'Q'` | 按季 |
| `'Y'` | 按年 |

### 模式 ③：处理缺失值 + 筛选保存

> 覆盖题号：3

**核心框架：**
```
读文件 → drop_duplicates → fillna(均值) → 条件筛选 → to_csv
```

```python
import pandas as pd
df = pd.read_csv('students.csv')
df = df.drop_duplicates()                          # 去重
df['分数'] = df['分数'].fillna(df['分数'].mean())   # 均值填充缺失值
high = df[df['分数'] >= 90]                         # 条件筛选
high.to_csv('high_score.csv', index=False)
```

| 缺失值操作 | 函数 |
|-----------|------|
| 删除含缺失值的行 | `df.dropna()` |
| 填充缺失值 | `df.fillna(值)` |
| 判断哪些是缺失 | `df.isna()` |

### 模式 ④：合并 + 分组求和

> 覆盖题号：5

**核心框架：**
```
pd.merge(左表, 右表, on='键', how='inner') → groupby → sum
```

```python
merged = pd.merge(df1, df2, on='学号', how='inner')    # 内连接合并
result = merged.groupby(['学号', '姓名'])['分数'].sum().reset_index()  # 分组求和
```

| 合并方式 | 含义 |
|----------|------|
| `how='inner'` | 内连接（取交集） |
| `how='left'` | 左连接 |
| `how='outer'` | 全外连接 |

## 三、网络爬虫（第 14—17 题）——2 种核心模式

### 模式 ①：requests + BeautifulSoup（静态页面爬取）

> 覆盖题号：15、16

**核心框架 4 步走：**
```
① requests.get(url, headers) 发请求
② BeautifulSoup(response.text, 'html.parser') 解析
③ soup.find_all() / soup.select() 提取数据
④ 保存到文件
```

**极简框架：**
```python
import requests
from bs4 import BeautifulSoup

headers = {'User-Agent': 'Mozilla/5.0'}
response = requests.get(url, headers=headers, timeout=10)
soup = BeautifulSoup(response.text, 'html.parser')

# 提取：用 find_all 或 select
items = soup.find_all('a', class_='类名')        # 按标签+类名
# 或
items = soup.select('div.item span.title')        # CSS选择器

for item in items:
    text = item.get_text()                        # 取文本
    link = item.get('href')                       # 取属性

# 保存
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write(...)
```

**关键方法速记：**

| 目的 | 方法 |
|------|------|
| 找所有匹配 | `soup.find_all('标签', class_='类名')` |
| CSS选择器 | `soup.select('CSS选择器')` |
| 取文本 | `.get_text()` / `.text` |
| 取属性 | `.get('href')` / `.get('src')` |

## 四、数据库操作（第 18—19 题）——2 种模式

### 模式 ①：pymysql 原生操作（题18）

**核心框架 6 步：**
```python
import pymysql
# ① 连接
conn = pymysql.connect(host='localhost', user='root', password='123456', db='school', charset='utf8mb4')
cursor = conn.cursor()
# ② 建表
cursor.execute('''CREATE TABLE IF NOT EXISTS 表名 (id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(20), score INT)''')
# ③ 插入
cursor.execute('INSERT INTO 表名(name, score) VALUES (%s, %s)', ('张三', 88))
# 或批量
cursor.executemany('INSERT INTO 表名(name, score) VALUES (%s, %s)', [('张三',88), ('李四',92)])
# ④ 提交
conn.commit()
# ⑤ 查询
cursor.execute('SELECT * FROM 表名')
rows = cursor.fetchall()
# ⑥ 关闭
cursor.close(); conn.close()
```

## 五、综合题（第 20 题）——四大步骤框架

> 把前面所有技能串起来的一道题，按这 4 步写框架即可：

```
① 数据清洗：drop_duplicates → dropna → pd.to_datetime → 提取月份
② 分组汇总：groupby + sum → pivot 生成透视表
③ 可视化：  plt.plot 各产品折线（循环+pivot.columns）
④ 保存结果：to_csv
```

**极简框架：**
```python
import pandas as pd
import matplotlib.pyplot as plt

# ① 清洗
df = pd.read_csv('sales.csv')
df = df.drop_duplicates().dropna()
df['日期'] = pd.to_datetime(df['日期'])
df['月份'] = df['日期'].dt.to_period('M').astype(str)

# ② 汇总
summary = df.groupby(['月份', '产品'])['销售额'].sum().reset_index()
pivot = summary.pivot(index='月份', columns='产品', values='销售额').fillna(0)

# ③ 绘图
for product in pivot.columns:
    plt.plot(pivot.index, pivot[product], marker='o', label=product)

# ④ 保存
pivot.to_csv('monthly_product_sales.csv', encoding='utf-8-sig')
plt.show()
```
