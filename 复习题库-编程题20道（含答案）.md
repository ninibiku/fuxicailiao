### 2. 使用Pandas库处理时间序列数据，完成以下任务：(1)读取"sales_time_series.csv"文件(包含"日期""销售额"字段);(2)将"日期"列转换为datetime格式;(3)按月份分组，计算每月的总销售额和平均销售额;(4)绘制时间序列折线图，显示销售额变化趋势;(5)保存处理后的数据到"monthly_sales.csv"。

monthly.to_csv('monthly_sales.csv')

### 14. 编写一个爬虫，模拟登录一个简单的网页（假设登录页面为https://www.loginpage.com，用户名输入框的name为username，密码输入框的name为password，登录按钮的name为submit），登录成功后获取登录后的页面内容。
**参考答案：**
```python
import requests

def login_and_get_content(url, username, password):
    login_data = {
        'username': username,
        'password': password,
        'submit': '登录'
    }
    with requests.Session() as session:
        try:
            response = session.post(url, data=login_data)
            response.raise_for_status()
            return response.text
        except requests.RequestException as e:
            print(f"请求出错: {e}")
            return None

url = 'https://www.loginpage.com'
print(login_and_get_content(url, 'your_username', 'your_password'))
```

### 15. 爬取一个新闻网站（如https://www.newswebsite.com）首页的所有新闻标题和链接，并将其保存到一个news.txt文件中。
**参考答案：**
```python
import requests
from bs4 import BeautifulSoup

def crawl_news(url):
    try:
        response = requests.get(url)
        response.raise_for_status()
        soup = BeautifulSoup(response.text, 'html.parser')
        news_items = []
        for news in soup.find_all('a', class_='news-link'):
            title = news.get_text()
            link = news.get('href')
            news_items.append(f"标题: {title}, 链接: {link}\n")
        with open('news.txt', 'w', encoding='utf-8') as f:
            f.writelines(news_items)
    except requests.RequestException as e:
        print(f"请求出错: {e}")

crawl_news('https://www.newswebsite.com')
```

### 16. 编写爬虫爬取豆瓣电影 Top250 第一页的电影名称与评分（假设条目位于 div.item 中，名称在 span.title，评分在 span.rating_num），结果保存到 douban.csv。
**参考答案：**
```python
import requests
import csv
from bs4 import BeautifulSoup

url = 'https://movie.douban.com/top250'
headers = {'User-Agent': 'Mozilla/5.0'}

response = requests.get(url, headers=headers, timeout=10)
response.raise_for_status()
soup = BeautifulSoup(response.text, 'html.parser')

rows = [('电影名称', '评分')]
for item in soup.select('div.item'):
    title = item.select_one('span.title').get_text(strip=True)
    score = item.select_one('span.rating_num').get_text(strip=True)
    rows.append((title, score))

with open('douban.csv', 'w', newline='', encoding='utf-8-sig') as f:
    csv.writer(f).writerows(rows)
```

### 19. 使用 Pandas 从 MySQL 数据库 sales 库的 orders 表读取数据（含 amount 字段），计算总订单金额，并把每位客户的累计金额写入新表 customer_total。

## 五、综合应用（第 20 题）

### 20. 综合编程：从 CSV 读入销售数据（sales.csv，含"日期""产品""销售额"），完成：(1) 数据清洗（去重、删除缺失值、日期类型转换）；(2) 按月份与产品统计销售额；(3) 绘制各产品销售额随时间变化的折线图（同一坐标系，不同颜色）；(4) 把汇总结果保存到 monthly_product_sales.csv。
**参考答案：**
```python
import pandas as pd
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# (1) 读取与清洗
df = pd.read_csv('sales.csv')
df = df.drop_duplicates().dropna()
df['日期'] = pd.to_datetime(df['日期'])
df['月份'] = df['日期'].dt.to_period('M').astype(str)

# (2) 按月份与产品分组
summary = df.groupby(['月份', '产品'])['销售额'].sum().reset_index()
pivot = summary.pivot(index='月份', columns='产品', values='销售额').fillna(0)

# (3) 折线图
plt.figure(figsize=(12, 6))
for product in pivot.columns:
    plt.plot(pivot.index, pivot[product], marker='o', label=product)
plt.title('各产品月度销售额变化趋势')
plt.xlabel('月份')
plt.ylabel('销售额')
plt.xticks(rotation=45)
plt.legend()
plt.grid(True, linestyle='--', alpha=0.5)
plt.tight_layout()
plt.show()

# (4) 保存
pivot.to_csv('monthly_product_sales.csv', encoding='utf-8-sig')
```
